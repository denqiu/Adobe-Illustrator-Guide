A Beginner's Guide to Adobe Illustrator

In this guide, we will learn about some of the basic tools and how they are used in Adobe Illustrator.

What is Adobe Illustrator?

Adobe Illustrator is a Creative Cloud software application for creating drawings, illustrations, and artwork using a Windows or MacOS computer. Illustrator is widely used by graphic designers, web designers, visual artists, and more throughout the world to create high quality artwork. Illustrator contains many sophisticated tools that help reduce the time needed to create the illustrations, such as the import function, which allows the user to import photographs and/or images that help guide the user when creating the illustration.

Tools in Adobe Illustrator

In Adobe Illustrator, there are two types of toolbars:

  • Basic - This toolbar appears by default and includes a selective list of tools that you will frequently need in Illustrator. Click on the Edit Toolbar icon at the bottom of the toolbar to view the All Tools drawer containing all the tools available in Illustrator.
  • Advanced - This toolbar contains all the tools available in Illustrator.
  • To switch between toolbars, do one of the following:
  • Choose Windows >> Toolbars.
  • Select the toolbar through the flyout menu inside the All Tools drawer.

We will cover several most frequently used tools in the Illustrator toolbar:

*NOTE: Right-clicking the tools with an arrow at the bottom right corner will allow the user to see more related tools, however we will not be covering them in this guide.*

  • Pen Tool

    It's shortcut key is (P), or [shift + P]. The Pen tool uses anchor points to draw straight and curved lines to create objects. Note that while both Adobe Photoshop and Adobe Illustrator has this tool, Illustrator's pen tool features anchor guidelines to help create straight lines and align with a particular straight part of an object.

  • Rectangle Tool

    It's shortcut key is (M), or [shift + M]. The Rectangle tool draws squares and rectangles.
    • To draw a rectangle, drag diagonally until the rectangle reaches the desired size.
    • To draw a square, hold down the Shift key and drag diagonally until the square reaches the desired size.
    • To create a square or rectangle using values, click anywhere you want without dragging to open up a pop up box. Then specify a width and height and click OK.

  • Eyedropper Tool

    It's shortcut key is (I), or [shift + I]. The Eyedropper tool is a very useful tool that samples the desired color of an object and applies the sampled color to another object's color, type, and appearance attributes. The user can click on the desired object's color to get the desired sample color.
